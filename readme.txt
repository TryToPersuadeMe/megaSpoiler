1. Классы в верстке HTML можно спокойно менять. Всё взаимодействие осуществляется через data атрибуты и кастомные классы, которые устанавливаются через js. Весь интеркатив через js

!! Исключение составляют классы accordion и accordion__сell !!. Они используются только в функции setupToggleDataset

2. Вложенность у файлов абсолютно одинаковая. Можно всё сделать через цикл. 
3. Для добавления текста, который бы появлялся при наведении достаточно интегрировать cell__note перед cell__container. Сделано через css
4. Прогресс бары обновляются только при первом рендере страницы


для работы с исходниками необходимо установить пакеты через node js командой 
npm i

после чего проверить, совпадает ли название папки с решеткой с названием, указанным в файле gulpfile в переменной let source_folder; ( = "#list" в данном случае) 
ввести команду gulp в node js
